`@afetcan/APÄ°` It is the package containing only api and services without the server package. We use the DB algorithms from the `@afetcan/Storage` package into service packages. All the necessary structures on the client side come from here. If we wish, we can expand this api and use it in a different way.

## NPM


### Setup
```bash
pnpm install @afetcan/api
```

### Usage
```ts
import { ... } from '@afetcan/api' 
```

## Requirements

- [Node.js](https://nodejs.org/en/) 18.0.0+
- [pnpm](https://pnpm.io/) 7.26.3+
